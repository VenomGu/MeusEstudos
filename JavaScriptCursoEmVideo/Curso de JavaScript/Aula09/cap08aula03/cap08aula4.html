Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Usando a tag code fica muito mais limpo a leitura, para escrever códigos dentro de um texto, fica muito melhor, como no exemplo assima.

Exemplo de linguagem em Python

num = int(input('digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

A tag code não mantem a intentação do código, mas se você usar a tag pre antes de code, ele irá manter a indentação como na cópia abaixo

        
    num = int(input('digite um número'))
    if num % 2 == 0:
            print(f'O número {num} é PAR')
    else:
            print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Para iminuir esses expaços, caso você queira, selecione a área que você quer e segure shift + tab e se quiser jogar para frente da área selecionada, use apenas tab.
